사용 방법

  1. 표준 양식 CSV 다운로드 후 Excel/메모장에서 편집
  2. 한 행에 1건. 일자는 YYYY-MM-DD 형식.
  3. 거래처코드는 비워두면 거래처명 그대로 저장됩니다 (DIM_거래처와 자동 매칭은 추후).
  4. 제품코드를 비우면 품명을 보고 자동 매핑 룰(32_제품매핑)이 적용됩니다.
  5. 업로드 → 미리보기 → 문제 없으면 "적용" 버튼.
