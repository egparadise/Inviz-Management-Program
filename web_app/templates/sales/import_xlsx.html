사용 방법

  1. 📊 표준 Excel 양식 다운로드 후 데이터 입력
  2. 양식의 노랑색 예시 행은 삭제하고 4행부터 매출 데이터를 입력
  3. 일자 형식: YYYY-MM-DD (예: 2026-06-01) — Excel 날짜 자동 인식
  4. 매출유형 컬럼은 드롭다운 (정기/신규/일회성/기타)
  5. 제품코드 비우면 품명에서 자동 매핑, 거래처코드 비우면 거래처명 그대로 등록
  6. 업로드 → 시트 선택 → 미리보기 → "적용"
