# assets — 배포 시 수동 배치 파일

## hometax_bulk_template.xls (홈택스 일괄발급 공식 양식)
`/tax/hometax-upload` 기능에 필요. 정책상 xls는 git에 커밋하지 않으므로 배포 시 직접 배치:

1. 홈택스 → 전자세금계산서 → 일괄발급 → **세금계산서등록양식(일반).xls** 다운로드
2. 파일이 기본 암호로 보호된 경우 해제 (Excel에서 열어 다른 이름으로 저장하면 해제됨)
3. 이 폴더에 `hometax_bulk_template.xls` 이름으로 저장

파일이 없으면 홈택스 업로드 화면에 안내 메시지가 표시되고 다운로드 버튼이 비활성화됩니다.
