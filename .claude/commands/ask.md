인비즈 경영관리 데이터에 대해 질문합니다.

질문: $ARGUMENTS

처리:
1. `inviz-business-query` 스킬 절차에 따라 키워드 → 의도 분류
2. fast match 가능하면 즉시 SQL (밀리초)
3. 모호하면 RAG 검색 (벡터 DB top-K)
4. 필요 시 LLM 컨텍스트 증강

답변 구조:
- 핵심 수치 (콤마·단위)
- 출처 (어느 테이블 / 어느 청크)
- 해당 페이지 링크 (예: /sales?year=2024)

- "올해 KPI 보여줘"
- "써밋영상의원 매출 누계"
- "30일 안에 만료되는 인증서"
